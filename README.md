## 🚀 Highlighted Projects

### 📦 [School Warehouse API](https://github.com/alxprgs/SCHOOL_WAREHOUSE_API)
An asynchronous enterprise-grade school asset and inventory tracking management system.
* **Stack:** FastAPI (Python 3.13), FastMCP, React 19 + Vite 7, SQLite/PostgreSQL, Docker.
* **Features:** AI integration via Model Context Protocol (MCP), OAuth 2.0 (fail-closed architecture), TOTP MFA, dynamic QR-code generation, and a fully automated GitHub Actions CI/CD pipeline deploying via SSH with integrated production-like smoke tests.

### 🏭 [ASFES Ecosystem](https://github.com/alxprgs/ASFES-MULTIPLEX)
A custom-built modular ecosystem designed to handle smart warehouse infrastructure and inventory tracking.
* **Stack:** FastAPI, MongoDB, Vanilla JS, WebSockets.
* **Features:** Real-time object detection via WebSocket camera streams, automated low-stock triggers with styled email reporting, and RBAC security. Hosted on a personalized domestic infrastructure.

### 🤖 [Arduino Motor Bridge (RTC_RDP)](https://github.com/alxprgs/RTC_RDP_Server)
A full-stack hardware bridge server for low-latency robotic platform controls.
* **Stack:** FastAPI, C++ (Arduino), `pyserial`, `psutil`.
* **Features:** Manages up to 5 validation-clamped servos via HTTP/WS, features software-level Watchdogs, an emergency actuator shutdown (E-STOP mechanism), servo slew-rate smoothing, and live host/microcontroller telemetry streams.

### 🛸 [UAV Delivery API](https://github.com/alxprgs/UAV-Delivery-API)
A logistics management REST API built to handle cargo delivery using unmanned aerial vehicles (drones).
* **Stack:** FastAPI, Motor (MongoDB), SQLAlchemy (MySQL), Redis (caching), YooKassa SDK integrations.
